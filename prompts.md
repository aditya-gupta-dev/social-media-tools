## initial app idea 

app-name = social-media-tools 

app idea=create a golang TUI application that initially copies the text from the clipboard then uses yt-dlp installed on the system to do three things first, either download thumbnail, download video, download audio. pick one and a progress bar will show with all the logs. at the bottom. If a valid youtube url, is not found then add a keyboard shortcut(n) to enter that url manually. i'll add support for more social media platforms so archtect the app accordingly. Storage space for all the downloads. Audio files->~/Music/$app-name, videos files->~/Videos/$app-name, Thumbnails /Pictures/$app-name. Download assets in the PWD if keyboard key C is pressed before the installation 

design theme = use the @apple.md theme 

developer instructions = For every social media there should be a seperate folder. like everything related to youtube should be in youtube/ folder. use bubble tea, lipgloss whatever best available to implement the design. You don't need to exactly follow the design just skeleton, curves, styles, colors will do it.
